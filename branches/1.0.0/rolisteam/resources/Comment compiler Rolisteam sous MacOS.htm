Comment compiler Rolistik sous MacOS (sur machine MacIntel)?

Vous devez avoir installé Xcode 3.0.

Téléchargez les sources de Qt 4.3.3 ici : http://trolltech.com/developer/downloads/qt/mac

Décompressez les sources dans un répertoire temporaire. Ouvrez un terminal, rendez-vous dans le répertoire temporaire et compilez Qt :

./configure -universal -static -no-openssl
make
sudo make install

Ajoutez le chemin de Qt au PATH, en rajoutant les lignes suivantes dans le fichier .profile (créez-le s'il n'existe pas, à la racine du répertoire personnel) :

PATH=$PATH:/usr/local/Trolltech/Qt-4.3.3/bin
export PATH

Fermez et rouvrez un terminal.

Dans les sources de Rolistik, renommez le fichier Makefile_MacOS en Makefile.

Compilez Rolistik en tapant make.

 

Les sources sont fournies avec le "Bundle" Rolistik.app contenant déjà l'icône ainsi que la documentation associée au logiciel.
